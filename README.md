# 📌 Project Overview

**Laptop Price Predictor 2026** is a **Machine Learning based web application** that predicts the estimated price of a laptop using different hardware and software specifications.

The application is built with **Python**, **Scikit-learn**, and **Streamlit** to provide a fast, interactive, and user-friendly experience.

Users can select laptop configurations such as:
- Brand
- Processor
- RAM
- Storage
- GPU
- Display Type
- Operating System

The trained ML model instantly predicts the estimated laptop price.
